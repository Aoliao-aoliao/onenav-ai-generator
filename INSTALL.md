# OneNav AI Generator 安装指南

## 快速安装

### 步骤1：上传插件文件

将整个插件文件夹上传到您的WordPress网站的 `/wp-content/plugins/` 目录下。

### 步骤2：激活插件

1. 登录WordPress后台
2. 进入"插件"页面
3. 找到"OneNav AI Generator"并点击"激活"

### 步骤3：配置插件

1. 进入"设置" > "OneNav AI Generator"
2. 填写DeepSeek API密钥
3. 根据需要调整其他设置
4. 点击"保存更改"

## 详细配置

### 获取DeepSeek API密钥

1. 访问 [DeepSeek官网](https://platform.deepseek.com/)
2. 注册账号并登录
3. 进入API管理页面
4. 创建新的API密钥
5. 复制密钥到插件设置中

### 推荐设置

- **最大生成字数**：2000（适合大多数网站介绍）
- **内容结构**：保持默认或根据需要自定义
- **自定义提示词**：可以根据您的网站类型调整

## 使用方法

### 单个网址生成

1. 编辑或新建网址文章
2. 填写网站基本信息
3. 点击"生成AI介绍"按钮
4. 等待生成完成

### 批量生成

1. 进入"工具" > "批量AI生成"
2. 点击"开始批量生成"
3. 等待处理完成

## 故障排除

### 常见问题

1. **插件激活失败**
   - 检查PHP版本是否为7.4+
   - 确认WordPress版本为5.0+

2. **生成失败**
   - 检查API密钥是否正确
   - 测试网络连接
   - 查看WordPress错误日志

3. **找不到生成按钮**
   - 确认文章类型为"sites"
   - 检查是否为OneNav主题

### 技术要求

- WordPress 5.0+
- PHP 7.4+
- OneNav主题
- 服务器能访问外网（DeepSeek API）

## 注意事项

1. 使用前请备份网站数据
2. API调用可能产生费用
3. 生成内容建议人工审核后发布

如有问题，请查看README.md文件获取更多帮助。
